SPREAD & REST OPERATORS

Merge two arrays using spread operator

// Arrays to merge const array1 = [1, 2, 3]; const array2 = [4, 5, 6]; // Merge using spread operator const mergedArray = [...array1, ...array2]; console.log(mergedArray); // Output: [1, 2, 3, 4, 5, 6]

Function using rest operator

// Function using rest operator function sum(...numbers) { return numbers.reduce((acc, num) => acc + num, 0); } // Call with multiple arguments console.log(sum(1, 2, 3)); // Output: 6 console.log(sum(10, 20, 30, 40)); // Output: 100

ðŸ’¡ This demonstrates using the rest operator to accept unlimited arguments The rest operator (...) allows a function to accept an indefinite number of arguments as an array

Key Points to Remember

  • Spread operator (...): Used to expand elements of an iterable (arrays, strings, etc.)
  • Rest operator (...): Used to collect multiple elements into an array
  • Both operators use the same symbol but have different contexts and purposes
  • Spread is used in array literals, function calls, and object literals
  • Rest is used in function parameter lists
